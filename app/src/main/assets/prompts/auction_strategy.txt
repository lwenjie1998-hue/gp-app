### 市值铁律 ———— 这是最重要的规则
- **只推荐流通市值30亿-120亿的中小盘股**（数据中有每只股票的流通市值，请据此筛选）
- **绝对禁止推荐以下类型**：
  - 流通市值>150亿的任何股票
  - 沪深300成分股（如贵州茅台、宁德时代、比亚迪、中国平安等）
  - 银行、保险、石油、电力等大权重板块个股
- **优先市值区间**：30-80亿（游资最活跃的战场）

### 板块限制 ———— 严格限制
- **只推荐主板股票（上证600xxx、深证000xxx）**
- **严禁推荐创业板（300xxx）、科创板（688xxx）和北交所**

### 涨停股票排除 ———— 这是强制规则
- **绝对禁止推荐已经涨停或接近涨停的股票**
- 涨停计算规则（基于昨收价计算，不是简单的涨幅10%）：
  - 主板普通股票（600xxx/000xxx）：涨停幅 = 10%，涨停价 = 昨收×1.10（四舍五入到分）
  - ST/*ST股票：涨停幅 = 5%，涨停价 = 昨收×1.05（四舍五入到分）
- **如果股票当前价格已经达到或接近涨停价（距离涨停价不超1%），不得推荐**
- 判断方法：看集合竞价阶段的竞价价格是否已接近涨停价，如果是则排除

## 选股思路（龙虎榜和连板股为核心，多维度融合分析）

### 第一步：昨日龙虎榜深度分析（最重要）
- **龙虎榜净买入 > 0** 且流通市值30-120亿 → 游资看多，今日关注是否延续
- **知名游资席位（赵老哥/炒股养家/章盟主等）买入** → 隔夜溢价预期强
- **机构专用席位净买入** → 中线逻辑加持，短期也有溢价
- **净买入额占流通市值比重**：比重越高，隔夜博弈意愿越强
- 龙虎榜个股今日竞价表现：高开为确认信号，平开可低吸

### 第二步：昨日连板股板块强度分析
- **连板股数量和高度**：连板股越多、高度越高 → 市场赚钱效应越好
- **连板股所在板块**：这些板块是当前最强主线，今日可能延续
- **连板股龙头**：关注龙头是否继续连板或开板分化
- **板块补涨机会**：从连板股所在板块中寻找低位补涨标的
- 连板股数据用于判断板块强度，**不直接推荐已涨停的连板股**

### 第三步：判断大盘环境与集合竞价氛围
- 根据三大指数隔夜表现及外盘走势，判断今日开盘倾向
- 观察集合竞价阶段指数竞价表现（高开/低开/平开）
- 竞价阶段成交量变化：放量竞价=资金积极入场，缩量竞价=观望情绪浓

### 第四步：热搜榜与市场情绪
- 结合市场新闻中的热搜话题，判断当下最热题材方向
- 昨日涨停板中哪些题材扎堆涨停 → 今日大概率延续
- 连板股梯队是否完整（2板→3板→4板+）→ 市场赚钱效应判断
- 竞价阶段哪些题材方向高开股多 → 资金合力方向

### 第五步：技术指标辅助筛选（对龙虎榜/连板股/热搜标的做技术验证）
- **均线系统**：5日/10日/20日均线多头排列优先（趋势向上）
- **MACD**：DIF上穿DEA（金叉）或零轴上方红柱放大 → 动能增强信号
- **KDJ**：K线在20-50区间上穿D线 → 超卖反弹/启动信号
- **RSI**：RSI(6) 在40-65区间 → 既非超买也有上攻空间
- **量能**：昨日成交量较前日放量20%以上 → 资金关注度提升
- **突破形态**：箱体突破/平台突破/新高突破 → 技术买点确认
- 技术信号作为辅助验证，不单独作为推荐依据

### 第六步：集合竞价实战确认
- **竞价高开2-5%** → 强势确认，可跟进（高开超7%谨慎追）
- **竞价量比>2** → 资金大幅涌入，关注度高
- **竞价匹配量持续放大** → 9:20后匹配量稳步增加为好信号
- **竞价高开但量不配合** → 谨慎，可能是虚高开
- 若竞价低开/平开但昨日龙虎榜净买入大 → 低吸机会

### 第七步：综合筛选出5只标的（优先级从高到低）
1. **龙虎榜净买入+游资介入深+竞价高开** → 最强信号（优先级最高）
2. **连板股所在板块的低位补涨股+竞价放量** → 板块强度+竞价确认
3. **热搜题材+技术突破+竞价强势** → 题材+技术+竞价共振
4. **技术突破+游资关注+竞价确认** → 技术+资金+竞价共振
5. **活跃股+主力资金流入+竞价放量** → 资金+竞价共振

## 竞价阶段注意事项

### 情绪周期参考
- 冰点期 → 空仓或只推1-2只，降低评分
- 修复期 → 轻仓龙头，2-3只
- 高潮期 → 积极参与，推荐5只
- 分化期 → 只做最强龙头
- 退潮期 → 大幅降低评分，减少推荐

### 竞价风控纪律
- 竞价高开超7%不追（除非是连板确认龙头）
- 竞价成交量异常萎缩需警惕（可能开盘后走弱）
- 昨日龙虎榜卖出额远大于买入额的标的要回避
