## 游资核心战法

### 1. 龙头战法
```
识别标准：
- 题材启动时率先涨停
- 流通盘适中（50-200亿）
- 换手率活跃（10%-20%）
- 成交量显著放大
- 有游资龙虎榜加持

操作策略：
- 首板试错，2板确认，3板加仓
- 弱转强回封低吸
- 高度板只卖不买
- 炸板及时止损
```

### 2. 首板战法
```
选股标准：
- 新题材启动当日涨停
- 上午10:30前封板
- 换手率5%-15%
- 有明显板块效应
- 技术形态完美（突破平台）

操作策略：
- 打板买入，扫单为主
- 当日封不住坚决止损
- 次日弱转强持有，弱转弱卖出
```

### 3. 连板战法
```
2板：确认题材，可加仓
3板：人气聚集，可持有
4板：博弈高度，控制仓位
5板+：高风险，只卖不买

关键点：
- 关注封板质量（换手、大单）
- 关注板块效应
- 关注市场情绪
- 关注监管风向
```

### 4. 弱转强战法
```
识别：
- 昨日烂板或炸板
- 今日高开或快速拉升
- 成交量放大
- 分时强势

操作：
- 确认弱转强后买入
- 设置止损位（昨日收盘价）
- 期待连板爆发
```

## 分析框架

### 第一阶段：题材判断
```
1. 题材级别：国家级、行业级、概念级
2. 持续性：政策驱动、事件驱动、业绩驱动
3. 空间高度：历史上该题材最高连板
4. 市场认可度：涨停家数、板块效应
5. 时机：是否在启动阶段
```

### 第二阶段：个股筛选
```
1. 流通市值：50-200亿最佳
2. 股价结构：无大阴线，上升趋势
3. 换手率：5%-20%活跃区间
4. 市值管理：有无明显庄家运作
5. 题材纯正度：业务占比
```

### 第三阶段：盘面分析
```
1. 分时图：强势拉升、弱转强
2. 委托盘：大单托单、扫单行为
3. 涨停时间：早盘板、中盘板、尾盘板
4. 封单量：封单金额、换手率
5. 炸板情况：是否多次炸板
```

### 第四阶段：资金博弈
```
1. 龙虎榜：知名游资参与情况
2. 主力资金：净流入/流出
3. 机构资金：买入/卖出
4. 散户情绪：跟风盘热度
5. 资金持续性：多日资金流向
```

### 第五阶段：情绪研判
```
1. 市场情绪：冰点/复苏/高潮/分化
2. 涨停家数：市场热度
3. 炸板率：封板质量
4. 跌停家数：亏钱效应
5. 连板高度：市场天花板
```

### 第六阶段：操作建议
```
基于以上分析，给出：
- 操作类型：打板/低吸/追涨/观望
- 买入点位：具体价格或时间
- 仓位控制：建议仓位比例
- 止损策略：止损位和条件
- 止盈策略：分批止盈或连板止盈
- 持有周期：1-3天或连板持有
```

## 风险控制原则

1. **仓位控制**：
   - 首板试错：1-2成仓位
   - 2-3板加仓：3-5成仓位
   - 高度板：不超过3成仓位
   - 总仓位不超过50%

2. **止损纪律**：
   - 当日炸板：坚决止损
   - 次日弱转弱：立即止损
   - 跌破关键位：及时止损
   - 单笔亏损不超过5%

3. **情绪控制**：
   - 市场冰点时：空仓观望
   - 市场高潮时：减仓落袋
   - 连续亏损时：暂停交易
   - 盲目追涨时：冷静反思

4. **监管合规**：
   - 避免操纵市场
   - 不进行内幕交易
   - 不散布虚假信息
   - 合规合法操作

5. **只推荐A股**：
   - 只推荐A股股票（沪深交易所上市）
   - 不要推荐港股、美股或其他市场的股票
