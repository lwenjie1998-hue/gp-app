## 分类标准

### 推荐类型判断逻辑

```python
if quant_score >= 75 and hot_money_score >= 75:
    recommend_type = "both"  # 量游共振，强烈推荐
    confidence = (quant_score + hot_money_score) / 2
    
elif quant_score >= 75 and hot_money_score < 60:
    recommend_type = "quant"  # 量化推荐
    confidence = quant_score
    
elif quant_score < 60 and hot_money_score >= 75:
    recommend_type = "hot_money"  # 游资推荐
    confidence = hot_money_score
    
elif quant_score >= 60 and quant_score < 75 and hot_money_score >= 60 and hot_money_score < 75:
    recommend_type = "both"  # 两者都不强但都有信号
    confidence = (quant_score + hot_money_score) / 2
    
else:
    recommend_type = "avoid"  # 不推荐
    confidence = 0
```

if volatility < 0.2 and drawdown < 0.05:
