## 游资尾盘核心战法

### 1. 尾盘选股铁律
```
趋势为王：只做上升趋势中的回调低吸，不抄底下降趋势
龙头优先：尾盘买入必须是板块辨识度最高的龙头或准龙头
量价验证：尾盘放量拉升必须伴随大单净流入，散户跟风量不算
情绪判断：大盘情绪退潮期尾盘买入 = 接飞刀，坚决不做
```

### 2. 尾盘买入的五大逻辑
```
逻辑一：强势股尾盘回调低吸
- 全天走势强势（涨幅3%以上）但尾盘小幅回调
- 主力资金全天净流入，尾盘回调是洗盘不是出货
- 次日大概率高开继续冲高
- 最佳买点：14:30-14:45回调至均价线附近

逻辑二：涨停板炸板后的尾盘回封博弈
- 早盘/午盘封涨停后炸板回落
- 尾盘资金重新回封或接近涨停
- 说明分歧后达成一致，次日溢价高
- 风控：必须有大单回封信号才介入

逻辑三：板块异动尾盘突击
- 14:00后某板块突然集体异动拉升
- 通常对应突发利好消息或游资集中扫货
- 买入板块中最强个股（涨幅最大/最先拉升）
- 博弈次日板块继续发酵

逻辑四：弱转强信号确认
- 全天低开低走但尾盘突然放量拉升翻红
- 从跌2%以上拉到平盘或微涨
- 分时图呈现V型反转
- 说明抛压释放后主力开始建仓，次日大概率高开

逻辑五：连板股尾盘确认买入
- 连板龙头尾盘仍稳稳封住涨停
- 封单量不减反增，资金锁仓意愿强
- 全天换手充分（10%-20%），筹码交换充分
- 次日大概率继续竞价高开，博弈连板溢价
```

### 3. 尾盘时间节奏
```
14:00-14:30（观察期）：
- 复盘全天走势，锁定候选标的
- 观察板块资金流向变化
- 确认龙头股封板质量

14:30-14:45（黄金狙击期）：
- 强势股回调低吸的最佳窗口
- 板块异动的主要爆发时间段
- 弱转强信号的确认时间

14:45-14:57（最终决策期）：
- 最终确认买入标的
- 挂单价格和仓位确定
- 此时不可撤单要谨慎

14:57-15:00（集合竞价）：
- 尾盘集合竞价阶段
- 关注大单突然涌入的标的
- 此阶段不可撤单，需提前计划好
```

### 4. 次日预期判断框架
```
高开预期评估：
- 隔夜利好消息面 → 高开幅度预判
- 龙虎榜知名游资介入 → 次日资金接力概率
- 板块整体强度 → 是否有板块效应加持
- 大盘情绪 → 系统性风险评估

次日操作预案：
- 高开3%以上：持有看能否冲板，冲板失败卖出
- 高开1-3%：观察前15分钟量价，强则持有弱则卖出
- 平开：观望30分钟，不涨即卖
- 低开：直接止损，不犹豫
```

### 5. 仓位与风控
```
单只标的：最多3成仓位
尾盘总买入：不超过5成仓位
隔夜最大亏损预期：单笔不超过3%（即高开失败的代价）
次日止损铁律：
- 集合竞价低开超3% → 开盘直接卖出
- 开盘15分钟内跌破昨日收盘价 → 止损
- 冲高回落跌破均价线 → 减仓
连续2笔隔夜亏损 → 暂停尾盘策略一天
```
