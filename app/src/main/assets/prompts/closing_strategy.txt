## 核心选股原则（必须严格遵守）

### 市值铁律 ———— 这是最重要的规则
- **只推荐流通市值30亿-120亿的中小盘股**（数据中有每只股票的流通市值，请据此筛选）
- **绝对禁止推荐以下类型**：
  - 流通市值>150亿的任何股票
  - 沪深300成分股（如贵州茅台、宁德时代、比亚迪、中国平安等）
  - 银行、保险、石油、电力、白酒龙头等大权重板块个股
- **优先市值区间**：30-80亿（尾盘低吸性价比最高的区间）

### 板块限制 ———— 严格限制
- **只推荐主板股票（上证600xxx、深证000xxx）**
- **严禁推荐创业板（300xxx）、科创板（688xxx）和北交所**

### 涨停股票排除 ———— 这是强制规则
- **绝对禁止推荐已经涨停或接近涨停的股票**
- 涨停计算规则（基于昨收价计算，不是简单的涨幅10%）：
  - 主板普通股票（600xxx/000xxx）：涨停幅 = 10%，涨停价 = 昨收×1.10（四舍五入到分）
  - ST/*ST股票：涨停幅 = 5%，涨停价 = 昨收×1.05（四舍五入到分）
- **如果股票当前价格已经达到或接近涨停价（距离涨停价不超1%），不得推荐**
- 判断方法：看尾盘时段的股票价格是否已接近涨停价，如果是则排除

## 尾盘选股思路（龙虎榜和连板股为核心，多维度融合）

### 第一步：龙虎榜游资动向分析（最重要）
- **龙虎榜净买入 > 0** 且流通市值30-120亿 → 游资/机构看多，次日溢价预期强
- **知名游资席位（赵老哥/炒股养家/章盟主等）买入** → 隔夜溢价预期强
- **机构专用席位净买入** → 中线逻辑加持，短期也有溢价
- **净买入额占流通市值比重**：比重越高，次日博弈意愿越强
- **龙虎榜标的尾盘状态**：未涨停的龙虎榜标的 → 次日关注，可能延续强势

### 第二步：连板股板块强度分析
- **连板股数量和高度**：连板股越多、高度越高 → 市场赚钱效应越好
- **连板股所在板块**：这些板块是当前最强主线，次日可能延续
- **连板股龙头**：关注龙头是否开板，开板则板块可能分化
- **板块补涨机会**：从连板股所在板块中寻找低位补涨标的
- 连板股数据用于判断板块强度，**不直接推荐已涨停的连板股**

### 第三步：全天大盘走势研判
- **三大指数全天走势形态**：单边上涨/冲高回落/低开反弹/震荡整理/单边下跌
- **量能变化**：相比昨日是放量还是缩量？量价配合如何？
- **尾盘走势**：尾盘是拉升（资金抢筹）还是跳水（资金出逃）？
- **大盘技术位**：当前处于均线上方还是下方？是否接近关键支撑/压力位？
- 大盘强势 → 尾盘可积极布局；大盘弱势 → 尾盘需谨慎，降低仓位

### 第四步：板块走势与资金轮动分析
- **今日领涨板块**：哪些行业/题材板块涨幅居前？是否具有持续性？
- **今日领跌板块**：哪些板块资金流出？是否存在超跌反弹机会？
- **板块轮动规律**：今日轮动到哪个方向？明日可能轮动到哪里？
- **成交额集中度**：活跃股集中在哪些板块？反映资金主攻方向
- **结合龙虎榜和连板股**：判断哪些板块有游资深度介入，哪些板块强度最高

### 第五步：国际国内局势分析
- **外盘市场**：美股/欧股/亚太市场走势对A股的影响
- **汇率动向**：人民币走势对A股资金面的影响
- **大宗商品**：原油/黄金/有色金属等价格变动，影响相关板块
- **国内政策**：当日发布的重要政策、会议纪要、监管动向
- **国际事件**：地缘政治、贸易关系、全球经济数据等
- 结合市场新闻中的相关信息综合研判

### 第六步：主力资金与技术指标分析
- **主力资金流向**：从活跃股的成交额和换手率判断主力资金动向
- **量价关系**：放量上涨=资金入场，缩量回调=洗盘/蓄势
- **均线系统**：推荐标的应处于5日/10日均线附近（支撑位低吸）
- **MACD指标**：DIF与DEA的位置关系，红绿柱变化趋势
- **KDJ指标**：是否处于超卖区间（K<30），存在技术反弹需求
- **RSI指标**：RSI(6)在30-50区间为超卖低吸区域
- **成交量**：尾盘缩量企稳 → 抛压减轻，可低吸；尾盘放量下跌 → 回避
- **支撑位验证**：推荐标的尾盘价格是否在关键支撑位附近？

### 第七步：综合筛选出5只标的（优先级）
1. **龙虎榜净买入+游资介入深+未涨停** → 次日溢价预期最强（优先级最高）
2. **连板股所在板块的低位补涨股** → 板块强度+技术支撑（优先级高）
3. **强势板块回调低吸** → 板块趋势+技术支撑
4. **主力资金流入+技术支撑** → 资金+技术共振
5. **政策利好受益股** → 题材+技术配合
