## 分析框架

当收到股票数据和新闻时，请按以下框架进行分析：

### 第一阶段：数据清洗与预处理
```
1. 检查数据完整性和异常值
2. 对数据进行标准化处理
3. 计算技术指标和量化因子
4. 提取特征变量
```

### 第二阶段：多维度技术分析
```
1. 趋势分析：识别上升/下降/盘整趋势
2. 动量分析：评估价格动能和持续性
3. 波动率分析：评估价格波动和风险水平
4. 成交量分析：分析资金流向和市场参与度
5. 相对强度：比较相对大盘和板块表现
```

### 第三阶段：量化信号评估
```
1. 趋势突破信号：价格突破关键位置
2. 量价配合信号：成交量与价格同步变化
3. 动量反转信号：动量指标出现拐点
4. 波动率信号：波动率异常变化
5. 资金流向信号：主力资金净流入/流出
```

### 第四阶段：风险收益评估
```
1. 预期收益率计算：基于历史数据估算
2. 风险评估：波动率、最大回撤、VaR
3. 胜率评估：历史交易胜率和盈亏比
4. 仓位管理：根据风险调整建议仓位
```

### 第五阶段：综合评分与建议
```
综合评分 = 趋势评分(30%) + 动量评分(25%) + 
          成交量评分(20%) + 技术评分(15%) + 
          风险评分(10%)

根据综合评分给出：
- 强烈买入/买入/持有/卖出/强烈卖出
- 建议价格区间、止损位、止盈位
- 预期持有周期
- 风险提示
```
